# Controls and Keyboard Shortcuts

Quick reference for frequently used shortcuts in the OasisW Editor.

## Camera Controls

The Editor's camera is controlled with the mouse and keyboard.

| Operation    | Controls                                                         |
| ------------ | ---------------------------------------------------------------- |
| Orbit        | Left Mouse Button + Drag                                         |
| Pan          | Middle Mouse Button + Drag<br />SHIFT + Left Mouse Button + Drag |
| Look Around  | Right Mouse Button + Drag                                        |
| Zoom / Dolly | Mouse Wheel                                                      |
| Move         | W-A-S-D                                                          |
| Move (fast)  | SHIFT + W-A-S-D                                                  |

## General Mouse Controls

| Operation                                              | Controls                  |
| ------------------------------------------------------ | ------------------------- |
| Select Entity                                          | Left Mouse Button         |
| Translate / Rotate / Scale Entity (according to gizmo) | Left Mouse Button + Drag  |

## Keyboard Shortcuts

Note: On macOS, use CMD instead of CTRL.

| Operation                  | Description                                                | Keyboard Shortcut            |
| -------------------------- | ---------------------------------------------------------- | ---------------------------- |
| New entity                 | Create a new entity as a child of the current selection    | Ctrl + E                     |
| Duplicate entity           | Duplicate the selected entity and all children             | Ctrl + D                     |
| Rename entity/asset        | Focus the name field of the selected entity or asset       | F2                           |
| Copy entity/asset          | Copy the currently selected entity/asset                   | Ctrl + C                     |
| Delete                     | Delete the current selection                               | Delete or Ctrl + Backspace   |
| Focus                      | Focus on the current entity selection in the 3D view       | F                            |
| Translate                  | Activate the translate gizmo in the 3D view                | 1                            |
| Rotate                     | Activate the rotate gizmo in the 3D view                   | 2                            |
| Scale                      | Activate the scale gizmo in the 3D view                    | 3                            |
| Toggle coord space         | Toggle Local/World coordinate space                        | L                            |
| Toggle all views           | Hide/Show all views (maximize viewport)                    | Space                        |
| Previous Selection         | Select previously selected items                           | Shift + Z                    |
| Undo                       | Undo last action                                           | Ctrl + Z                     |
| Redo                       | Redo last action                                           | Ctrl + Y or Ctrl + Shift + Z |
