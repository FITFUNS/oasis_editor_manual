# Interface

Learn about OasisW's basic interface and tools.

## Opening Scenes

When you open a project, a scene selection screen is displayed. Click on a scene from this screen to start working.

![Open Project Screen](/img/usage-guide/1_open_project.png)

Once you open a scene, you can start game development work using various tools and views.
