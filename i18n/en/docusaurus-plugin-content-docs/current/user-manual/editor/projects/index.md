## Overview

Projects are the foundation for everything you create in the OasisW Editor. A project contains all the assets, scenes, scripts, and settings needed to build your application or game. This section covers all aspects of project management, from creation to configuration.

## Project Management Topics

### Getting Started

- **[Creating Projects](/usage-guide/create-a-project)** - How to create new projects.

### Working with Projects

- **[Project Dashboard](/user-manual/editor/projects/dashboard)** - Understanding and navigating the project dashboard interface.
- **[Project Settings](/user-manual/editor/projects/settings)** - Configure your project's name, description, privacy, and branding.
- **[Team Management](/user-manual/editor/projects/team-management)** - Adding collaborators and managing permissions.
- **[Development Logs](/user-manual/editor/projects/dev-logs)** - Using dev logs for project updates and team communication.

### Advanced Management

- **[Backup and Archiving](/user-manual/editor/projects/backup-archiving)** - Protecting your projects with backups and archives.
- **[Ownership and Transfers](/user-manual/editor/projects/ownership-transfers)** - Transferring projects between users and organizations.

## Quick Actions

Once you understand the basics of project management, you can:

- Access the [Editor](/user-manual/editor/interface) to build your project.
- Test your project using the [Launch Page](/user-manual/editor/interface/launch-page).
- Use [Real-time Collaboration](/user-manual/editor/realtime-collaboration) to build with others.
- Use [Version Control](/user-manual/editor/version-control) to manage your development workflow.
