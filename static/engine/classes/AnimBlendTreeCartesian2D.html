    Class AnimBlendTreeCartesian2D

    An AnimBlendTree that calculates its weights using a 2D Cartesian algorithm based on the thesis http://runevision.com/thesis/rune_skovbo_johansen_thesis.pdf Chapter 6 Section 3.
