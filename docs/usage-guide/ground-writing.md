# 바닥면 만들기

OasisW에서 평면 오브젝트를 생성하여 게임의 바닥면(Ground)으로 설정하는 방법을 학습합니다.

## 바닥면 만들기

### 1단계: Hierarchy 뷰에서 Plane을 생성합니다.

1. **하이어라키 뷰**에서 **Add Entity**를 클릭합니다.
2. **3D** → **Plane**을 선택합니다.

![ground](/img/tutorial/2_1.png)

### 2단계: 크기 조정

**Scale** 속성을 사용하여 바닥면의 크기를 지정합니다.

![ground](/img/tutorial/2_2.png)
