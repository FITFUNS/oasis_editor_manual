## OasisW란 무엇인가요?

OasisW는 ✨ **웹 그래픽 생성 플랫폼** ✨입니다

OasisW는 인터랙티브하고 그래픽적인 웹 애플리케이션을 구축하기 위한 강력한 오픈소스 JavaScript 엔진을 제공합니다:

🦾 **OasisW Studio** - 모든 OasisW 애플리케이션을 구동하는 오픈소스 JavaScript 런타임
