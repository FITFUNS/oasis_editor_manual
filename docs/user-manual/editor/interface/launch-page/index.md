---
title: 실행 페이지
---

에디터의 뷰포트(Viewport) 는 애플리케이션을 시각적으로 구성하는 공간입니다. 여기서 Launch 버튼을 누르면 실행 페이지(Launch Page)가 열리며, 애플리케이션을 실제로 실행 해볼 수 있습니다.

![Launch Button](/img/user-manual/editor/launch-page/launch-button.png)

## 실행 페이지 사용하기

실행 페이지를 활성화하면 새 탭에서 열립니다. 에디터와 실행 페이지를 오가며 사용하는 것은 전혀 문제없습니다.
다만 화면이 넓거나 멀티 모니터 환경이라면, 실행 페이지 탭을 분리해 에디터와 나란히 배치하는 것을 권장합니다.

![Launch Page Side by Side](/img/user-manual/editor/launch-page/launch-page-side-by-side.png)

이렇게 하면 두 화면을 더 쉽고 빠르게 전환할 수 있습니다.

## 실시간 업데이트

실행 페이지의 핵심 기능 중 하나는 에디터와 실시간 연결(라이브 링크) 을 유지한다는 점입니다.
즉, 에디터에서 변경한 내용은 즉시 실행 페이지에 반영됩니다.

![Launch Page Live Link](/img/user-manual/editor/launch-page/launch-page-live-link.gif)

또한 에디터에 함께 접속해 있는 다른 사용자(쓰기 권한이 있는 경우)가 있다면, 해당 사용자의 수정 사항 역시 실행 페이지에 바로 반영됩니다.
