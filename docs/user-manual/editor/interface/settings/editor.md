기본적으로 에디터는 대부분의 사용자를 만족시킬 수 있는 설정으로 구성되어 있습니다.
하지만 특정 상황에서는 에디터의 기본 동작을 변경하고 싶을 수 있습니다.

:::note

이 설정들은 본인에게만 적용되며, 프로젝트 전체에 대해 전역적으로 사용됩니다.
또한 한 번 설정하면 여러 세션에 걸쳐 유지(sticky) 됩니다.

:::

`EDITOR` 섹션으로 이동한 뒤 패널을 확장하세요.

![Editor Settings](/img/user-manual/editor/interface/settings/editor.webp)

아래는 사용 가능한 설정 항목에 대한 설명입니다: 

## 설정 {#settings}

| 설정                    | 설명               |
| ---------------------- | ---------------------------------------------------- |
| **Grid Divisions**     | 각 수평 방향의 그리드 셀 개수를 지정합니다. 0으로 설정하면 그리드가 비활성화됩니다.    |
| **Grid Division Size** | 하나의 그리드 셀 크기를 지정합니다.                                         |
| **Snap**               | [기즈모](../viewport.md#gizmos) 스냅 간격을 설정합니다. 기즈모를 사용하는 동안 Shift 키를 누르거나 툴바의 Snap 토글을 사용하면 스냅을 활성화할 수 있습니다.                                  |
| **Camera Clip Near**   | 에디터 카메라의 Near 클립 값을 조정합니다. 이 설정은 게임에는 영향을 주지 않습니다.   |
| **Camera Clip Far**    | 에디터 카메라의 Far 클립 값을 조정합니다. 이 설정은 게임에는 영향을 주지 않습니다.    |
| **Zoom Sensitivity**   | 에디터 뷰포트에서의 줌 민감도를 조정합니다.                                   |
| **Clear Color**        | 에디터 카메라의 클리어 색상을 설정합니다. 이 설정은 게임에는 영향을 주지 않습니다.     |
| **Icons Size**         | 에디터 뷰포트에 표시되는 아이콘의 크기를 설정합니다.                            |
