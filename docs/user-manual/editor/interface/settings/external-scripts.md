**외부 스크립트** 설정은 프로젝트에서 외부 JavaScript 라이브러리를 로드하여 에디터와 런타임에서 사용할 수 있도록 합니다.

:::note

이 설정은 현재 활성화된 프로젝트 브랜치의 모든 사용자에게 영향을 줍니다.

:::

`EXTERNAL SCRIPTS` 섹션으로 이동한 뒤 패널을 펼치세요:

![외부 스크립트 설정](/img/user-manual/editor/interface/settings/external-scripts.png)

## 사용 방법

* 필요한 외부 스크립트 개수만큼 **URLs** 숫자를 입력합니다.
* 생성된 URL 입력 필드에 외부 라이브러리의 URL을 입력합니다.

이렇게 추가한 외부 라이브러리는 프로젝트에 로드되며, 에디터 및 애플리케이션 코드에서 바로 사용할 수 있습니다.

## 활용 예시

- 외부 수학/물리 라이브러리 로드
- 유틸리티 스크립트(CSV 파서, 압축 라이브러리 등) 사용
