---
title: 게시하기
---

게임을 제작한 후에는 사람들이 즐길 수 있도록 게시하고 싶을 것입니다! OasisW는 거의 모든 운영 체제를 대상으로 쉽게 게시할 수 있도록 도와줍니다.

게시하려면 프로젝트에 최소한 하나의 씬이 있어야 합니다.

## 새 빌드 게시하기 {#publishing-a-new-build}

* 프로젝트로 이동하여 에디터를 엽니다.
* 프로젝트 툴바에서 **Publish / Download** 버튼을 클릭합니다.

![Publish Button](/img/user-manual/editor/publishing/toolbar-publish.png)

* 게시 대화상자가 열립니다. 여기서 빌드를 게시하거나 자체 호스팅을 위해 빌드를 다운로드할 수 있습니다.

![Publish New Build](/img/user-manual/editor/publishing/dialog-publish-build.png)

* 새 빌드에 대한 세부 정보를 입력해야 합니다. 사용 가능한 필드는 다음과 같습니다:
  * **Project Image**: 원하는 경우 이미지를 클릭하여 다른 이미지를 업로드할 수 있습니다. 소셜 미디어에서 빌드를 공유할 때 사용됩니다.
  * **Title**: 빌드의 제목입니다.
  * **Description**: 빌드에 대한 설명입니다. 소셜 미디어에서 빌드 링크를 공유할 때 사용됩니다.
  * **Choose Scenes**: 목록에서 빌드에 포함할 씬을 선택합니다. 빌드에 포함된 경우 활성 배너 아이콘이 있는 씬이 빌드가 시작될 때 먼저 로드됩니다.
* **Publish to Campfire** 버튼을 클릭하고, 처리 과정이 완료될 때까지 기다리세요.
* 웹 앱이 이제 정식으로 출시되었습니다! 게시된 URL을 바로 공유해 보세요.
