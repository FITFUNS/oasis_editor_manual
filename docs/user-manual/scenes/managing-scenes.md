## 장면 다이얼로그 열기

**장면 이름 클릭** - 프로젝트 툴바에 표시된 현재 장면 이름을 클릭합니다.

![Scenes Menu](/img/user-manual/scenes/managing-scenes/scenes-viewport.jpg)

## 장면 다이얼로그 개요

![Scenes Menu](/img/user-manual/scenes/managing-scenes/scenes.jpg)

장면 다이얼로그에는 프로젝트에 포함된 모든 장면이 표시됩니다.

- **필터** - 검색 입력창을 사용하여 장면 이름으로 필터링할 수 있습니다.
- **장면 목록** - 장면은 알파벳 순으로 정렬되며, 장면 이름과 마지막 수정 날짜가 함께 표시됩니다.
- **현재 장면** - 현재 로드된 장면은 강조 표시됩니다.

## 장면 열기

목록에서 원하는 장면을 클릭하면 해당 장면이 열립니다.

## 새 장면 생성하기

1. 다이얼로그 하단의 **+ ADD NEW SCENE** 버튼을 클릭합니다.
2. 새 장면의 이름을 입력합니다.
3. **Create** 버튼을 클릭하면 새 장면이 생성되고 바로 열립니다.

## 장면 작업 메뉴

각 장면 오른쪽에 있는 드롭다운 버튼을 클릭하면 추가 작업을 수행할 수 있습니다.

![Scenes Menu](/img/user-manual/scenes/managing-scenes/scene-dropdown-menu.jpg)

| 작업 | 설명 |
| --- | --- |
| Duplicate Scene | 선택한 장면을 복사합니다. 복제된 장면은 자동으로 번호가 증가된 이름으로 생성됩니다. (예: `My Scene 2`) |
| Delete Scene | 확인 후 장면을 영구적으로 삭제합니다. |
